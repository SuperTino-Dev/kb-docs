# Oracle Edition-Based Redefinition (EBR) Lab

## What is EBR?

Edition-Based Redefinition (EBR) is an Oracle Database feature that allows multiple versions of application code to exist simultaneously.

This enables:

- Online upgrades
- Near zero downtime deployments
- Rolling application changes
- Safe PL/SQL migrations

Very common in:

- Oracle Fusion ERP
- Oracle E-Business Suite Online Patching
- SaaS applications

---

Package Body

PROCEDURE show_customer(p_id NUMBER) IS

    SELECT name

    WHERE customer_id = p_id;

Editioning Views
Why Editioning Views Exist

Tables are not editionable.

Oracle uses Editioning Views to abstract table structure changes.
